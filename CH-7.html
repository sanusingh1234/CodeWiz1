Collections In Python :

Python have four types of data collection.

a. List
b. Tuple
c. Set
d. Dictionary

List :

List is a collection of data type which is ordered and can be changed(mutable). And are written with square brackets around them. It allows same members(duplicate) in it.

Creating a List :-

Input :-

List1 = ["python", "java", "C++"]
print(List1)
print(type(List1))

Output :-

['python', 'java', 'C++']
class 'list'

Accessing Items in list

It is very easy to do this can be done by referring to their index number(index number refers to the position of the element in python it starts from 0 i.e the first element will have index no. of 0 and will continue till the last element). Example -

Input :-

print(List1[0])      # It will print the first element of the list as indexes starts from 0
print(List1[1])
print(List1[2])

Output :-

python
java
C++

Negative Indexing

Negative indexing starts from the end of a collection, here the index of the last element is -1 and tthe last second is -2 and so on. Example -

Input :-

List1 = ["python", "java", "C++"]
print(List1[-1])      # It will print the first element from the last of the list as negative indexes starts from -1 from last
print(List1[-2])
print(List1[-3])

Output :-

C++
java
python

Range Of Indexes

In python we can specify range of indexes by specifying the starting and ending index. Same as accessing elements but with a ending index, while both are seprated by a colon. Example -

Input :-

list_2 = ["hey", "hi", "hlo", "banana", "fruits", "mango", "got you"]
print(list_2[2:6])
print(list_2[:])

Output :-

['hlo', 'banana', 'fruits', 'mango']
['hey', 'hi', hlo', 'banana', 'fruits', 'mango', 'got you']

If you leave the first value empty then it will start from the first item and as same for the end value it will go to the end of the list.

Tuple

Tuple is a collection of data type which is ordered and can't be changed(immutable). And are written with round brackets around them. It allows same members(duplicate) in it. An example of a tuple is given below .

Input :-

tuple_a1 = ("python", "java", "C++")
print(type(tuple_a1))
print(tuple_a1)

Output :-

class 'tuple'
('python', 'java', 'C++')

Accessing items in tuples, and many other methods goes as same as lists.

Changing tuple values.

Since tuples are immutable in python so we cannot change a variable in tuple but this can be done by following a different approach. First typecast the tuple into a list, now we can change the value as python allows us to change the value of a list. Example given below.

Input

tuple_a1 = ("python", "java", "C++")
tuple_2 = list(tuple_a1)     #Converting to list so as to change a desired value
tuple_2[0] = "javascript"
tuple_a1 = tuple(tuple_2)     #Converting back list to tuple
print(tuple_a1)

Output :-

('javascript', 'java', 'C++')

SETS

A set is a collection which is unordered, and unindexed. In python sets are written with curly bracket around them. Creating a set is shown below.

Input :-

set_1 = {"dotted", "choclate", "strawberry"}
print(set_1)
print(type(set_1))

Output :-

{'dotted', 'choclate', 'strawberry'}
class 'set'

In sets you cannot print values by index values but you have to loop through it to print values.

Input :-

set_1 = {"dotted", "choclate", "strawberry'}

for x in set_1 :    #You can loop through the set items using a 'for' loop, or ask if a specified value is present in a set, by using the 'in' keyword.
        print(x)

Output :-

dotted
choclate
strawberry

You can use add(), update(), len(), remove(), discard(), pop(), union() to do operations on sets and there are many more functions availabe you can cehck them on python's official website.

DICTIONARY :

A dictionary is a collection which is unordered, mutable and indexed. Basically dictionaries in python are similar to dictionaries which you use, python dictionaries have keys and values corrsponding to them. In python dictionaries are written with curly brackets around them. Creating a dictionary is shown below.

Input :-

dict_1 = {"total": "200", "boys": "125", "girls": "75"}
print(dict_1)
print(type(dict_1))

Output :-

{'total': '200', 'boys': '125','girls': '75'}
class 'dict'

Collections in python have many functions or methods almost each of them have same way of syntax you can check all these on the python's official website, there you will find a lot of useful functions.
