String Literals

Strings in python are surrounded by single('') or double quotes("") while programming in python. By using print function we can print them.

Input :-

print('hi')

hi

String Operators

Basic Operator

Concatenation Operator

Two strings can be added using + operator.

Example:

>>> "Hello"+"CodeWiz"
'HelloCodeWiz'

Replication Operator

A string can be replication using * operator. It works when one operand is number and the other is string.

Example:

>>> "CodeWiz"*2
"CodeWizCodeWiz"

Using upper and lower functions on string

lower() function is used to convert a string into lower case word(string) whereas the upper() function is used to convert a string into upper case word(string). Or in short we can say that lower() non-capitalizes the whole string and upper() capitalizes the whole string.

Input :-

a = "HELLO THIS IS PYTHON"
b = "hello this is python"

print(a.lower())
print(b.upper())

Output :-

hello this is python
HELLO THIS IS PYTHON
