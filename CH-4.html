Numbers

In Python, Number data type is used to store numeric values.There are total 3 types of numbers in python. -

a. int
b. float
c. complex

Int

Int(integers) are whole numbers ranging from -ve infinity to +ve infinity, but without decimals. It means it has no fractional parts. For example: +34, -99, 67, etc.

Example:

Input :-

a = 89
b = -8338738738
print(type(a))
print(type(b))

Output :-

int
int

Float

Float(Floating Point Number), these are basically integers but with decimals. It means it has fractional part. In Python, Floting point numbers have precision of 15 digits.

Example:

Input :-

a = 6.9
b = 69.0
c = -6969.6969
d = -1.0
print(type(a))
print(type(b))
print(type(c))
print(type(d))

Output :-

float
float
float
float

Complex :

Complex numbers are of the form a+bi, where i is the imaginary number. i is euals to the square root of -1, or square if i is -1

Example:

v = 10j, b = -24j, x = 69+6j

You can find out real or imaginary part by using (variablename).real or (variablename).imag respectively.

Example:

>>> a=1.5+5.6i
>>> a.real
1.5
>>> a.imag
5.6

Implicit type conversion

It is a type of conversion performed by the compiler withot programmer's intefere. It is applied when in an operations the two operands are of different data types. It is also called Coercion.

Example:

>>> a=5
>>> b=8.0
>>> c=a+b
>>> print(c)
Now, the interpreter will convert int type to float type for the operation and print the output:
13.0

Explicit type conversion

It is the type of conversion performed by the programmer itself. Value of one data type is converted into another data type forcefully by the user.It is also called Type Casting.

Example:

>>> a=5
>>> print(float(a))
will print
5.0
