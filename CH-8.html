if, elif and else statement in Python

In python if, elif, else are used for decision making by using logical conditions from mathematics.

if

The 'if' is a keyword in python which saya if something is true then execute the following codes. We need an example to understand this.

Input :-

x = 40
y = 90100
if y > x :
      print(y, "is greater than", x)

Output :-

90100 is greater than 40

Logic behind this

Here we have used two variables, 'x' and 'y', which we have used as a part of if statement to test wether y is greater than x or not. As value of x is 40 and value of y is 90100. Since this is true therefore we got print the statement print on screen that is 90100 is greater than 40. Here colon are very necessary with them you will get error.

Indentation :

Putting 4 whitespaces after the next line of the conditional statement. This helps python to understand that we want to get in that statement and execute the code. In other languages tthis is done by using curly brackets. If you don't put these four blank spaces then you not be able to execute the conditions present in that statement .

elif

The 'elif' is a keyword in python which says if the above condition is not true then execute the next conditional statement. We need an example to understand this.

Input :-

x = 400
y = 90
if y > x :
      print(y, "is greater than", x)
elif x > y :
      print(x, "is greater than", y)

Output :-

90100 is less than 40

else

The 'else' keyword executes it's code when the previos 'else' and 'elif' statements are not true. We need an example to understand this.

Input :-

x = 90
y = 90
if y > x :
      print(y, "is greater than", x)
elif x > y :
      print(x, "is greater than", y)

else x == y :
      print(x, "is equal to", y)

Output :-

90 is equal to 90

pass statement in python

If you want the if statement to be empty then you have to use it becaues without pass statement python will give error. Example

Input :-

x = 90
y = 90
if y > x :
      pass
print("Hello World")

Output :-

Hello World
