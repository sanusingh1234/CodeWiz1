Print Statement In Python :

Print is an inbuilt function provided by python used to print any statement on output.It can print any value, variable, constant and list of variables.
Print Follows A Simple Syntax:

>>> print()

To print any value we just need to write print, and the open parenthesis write the value you want to print and next close the parenthesis.
Now We Are Ready To Print Our First Statement:
Just type the following command in your workspace and run the program:

>>> print("Hello World")

After running this program you will see a output which shows Hello World on the console.
As we all know that Hello World is a string so as to print this we need to put it between single or double quotes. If you don’t do that you will get error because the interpreter gets confused.

Printing a numeric Value :

To print numeric values you don’t need any quotes to put after and before the value. They can be print as shown below:

>>> print(9)

To print any variable, first you need to assign a value to the variable let say
>>> X = 78
>>> Y = “Learn to code”
Now you can easily print them by
>>> print(X, Y)

Source Code And Object Code :

The input which is written by a programmer is called Source code whereas the output from the console is called Object code.

Comments In Python :

Comments are additional readable information for pragrammers.Comments are lines of code in program which does not have any effect on Object code. It is generall used to clarify some lines of code like what this list is about and what this if statement do or what this loop will do.
It has 2 types:
Single line comment
Multi-line comments.

Single line Comment

A single line comment can be written by starting a line of code using #.
Example:
>>> #Main program
>>> #Program to calculate square

Multi-line Comment

A multiline comment can be written by starting and ending a line of code using “””.
They are used to clarify the code as shown below:
''' Program to find out the area and perimeter of
Circle
Rectangle
Square
This is what the program is'''

Dynamic vs Static Typing

Static Typing

When a variable is assigned to a value of one type then, it cannot be used to assign value of another type.

Dynamic Typing

When a variable is assigned to a value of one type then, the same variable can be used to assign value of anpther type. Python uses Dynamic typing. For example:
>>> a=10
>>> print(a)
>>> a="Hello"
>>> print(a)
Then the output will be :
10
Hello
