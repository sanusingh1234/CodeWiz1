Loops In Python

Python has two types of loop commands :

  while loops
  for loops

while Loop :

With the 'while' loop we can execute a set of statements as long as a condition is true. Example

Input :-

i = 50
while i < 10 :
      print(i)
      i = i + 1      #It will print i till it is less than 10 by adding 1 to next step

Output :-

5
6
7
8
9

for Loop :

A for loop is used to iterarting over a sequence (that is a list, tuple, dictionary, set, string).Under this we can execute a set of statements, once for each item in a sequence. Example

Input :-

hola = ["car", "spaceship", "submarine"]
for x in hola :
      print(x)

Output :-

car
spaceship
submarine

We can loop though a string also -

Input :-

for x in "python" :
      print(x)

Range function :

Input :-

for x in range(4,8) :
      print(x)    # It will print the numbers between 4 to 8

Output :-

4
5
6
7

The range() function incriments the sequence by 1 (it is default value). But we can change the incriment by adding a third prompt or parameter in range().Example

Input :-

for x in range(1, 20, 3):
      print(x)

Output :-

1
4
7
10
13
16
19

  You can alo use if, elif and else statements also in these loops.
