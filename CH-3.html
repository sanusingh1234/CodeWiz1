WHAT ARE VARIABLES IN PYTHON ?

Variables are reversed memory locations to store values. In simple you can think of them as a container to store values. Unlike other languages you don't need to declare that you are making a variable and even you don't need to declare it's type. Python is a object oriented programming language so everything is a object in it.

Creating Variables :

You can easily create a variable by assigning a value to it. String values can be declared by using single or double quotes.

Rules For Variables Name :-

1. It should start with a letter or undersocre(_) character.
2. You cannot start a variable with a number.
3. You can name a variable using only alpha-numeric value and underscore.
4. You cannot use python's reserved words to name like random, math because they are built-in modules of python.
5. In python variable's names are case sensetive for e.g - King,king,KING,kiNG,kinG these all are different names.

Assigning More Values To Variables And a Single Value To More Variables :

Python allows to assign more than one value to different variables in one line or more.For e.g :-

Input :-

a =, b=, c= "bot", 5, "every noob pro is a bot"
print(a)
print(b)
print(c)

This syntax will result in the following output :-

bot
5
every noob pro is a bot

You can assign a single value to more than 1 or more variables.This can be done by the following :-

Input :-

a = b = c = "CodeWiz!"
"And now if you print anyone of the above values you will get same result or output for each of these three values."
print(a)
print(b)
print(c)

The Output Will be :-

CodeWiz!
CodeWiz!
CodeWiz!

Adding Varibles :

You can add two or more variables in python. This can be done in the following way :-

Input :-

x = 5
y = 10
print(x + y)

Output will be :-

15
Similarly you can add two or more strings.For e.g-

Input :-

x = "Code"
y = "Wiz"
print(x + y)

Output :-

CodeWiz
