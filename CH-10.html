Pass statement :

'pass' staements are used to leave a conditional statement empty. If you leave empty any conditional statement without pass you will get an error but to oevercome or skip that error 'pass' is used in python. Example

Input :-

for x in [1,5,0] :
      pass
m = 24
if m > 50 :
      pass
print("huhh")

Output :-

huhh

Continue statement :

'continue' statements are used to continue to the next iteration by stoping the current one.Example

Input :-

i = 10
while i < 20 :
      i = i + 1
      if i == 15 :
            continue      #Here it will stop the loop on 15 and will again return to the while loop and will start printing from 16.
      print(i)

Output :-

11
12
13
14
16
17
18
19
20

Another Example :

Input :-

bot = ["bot425", "bot223", "bot_hlo"]
for x in bot :
      if x == "bot223" :
            continue
      print(x)

Output :-

bot425
bot_hlo

break Statement :

Same as continue but it will break and exit the loop. Example given below

Input :-

bot = ["bot425", "bot223", "bot_hlo"]
for x in bot :
      if x == "bot223" :
            break
      print(x)

Output :-

bot425
bot223
